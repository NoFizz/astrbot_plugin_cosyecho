# 声音复刻/音色克隆（TTS） - CosyVoice 插件

基于阿里云百炼 CosyVoice 声音复刻的 AstrBot 语音合成插件。LLM 回复自动转为语音发送，支持指令控制情感、方言、语速等效果，群聊/私聊独立白名单与触发概率，可选翻译后合成。

## 支持的模型

| 模型 | 说明 |
|------|------|
| `cosyvoice-v3.5-plus` | 支持任意指令控制合成效果 |
| `cosyvoice-v3.5-flash` | 轻量版，支持任意指令控制合成效果 |
| `cosyvoice-v3-plus` | 指令需遵循固定格式 |
| `cosyvoice-v3-flash` | 轻量版，指令需遵循固定格式 |

## 快速开始

1. 在 AstrBot 插件市场中安装本插件
2. 前往[声音复刻](https://bailian.console.aliyun.com/cn-beijing?tab=model#/efm/model_experience_center/voice?currentTab=voiceTts&primary=cloning&secondary=clone)创建复刻音色，获取音色 ID
3. 前往[API Key](https://bailian.console.aliyun.com/cn-beijing?tab=model#/api-key)获取阿里云百炼 API Key
4. 在插件配置中填入 API Key、模型、音色 ID 即可使用

## 配置说明

### 基础配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **API Key** | 阿里云百炼 API Key | 空 |
| **模型 ID** | CosyVoice 模型 | `cosyvoice-v3.5-plus` |
| **音色 ID** | 声音复刻音色 ID（需在百炼平台创建） | 空 |
| **超时时间** | API 请求超时（秒） | 20 |
| **音量/volume** | [0, 100]，50 为标准音量 | 50 |
| **语速/rate** | [0.5, 2.0]，1.0 为标准语速 | 1.0 |
| **音高/pitch** | [0.5, 2.0]，1.0 为自然音高 | 1.0 |
| **目标语言** | 优化指定语言的合成效果 | 中文 |
| **指令/instruction** | 控制方言、情感或角色等合成效果 | 空 |

### 触发控制

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **群聊语音开关** | 是否启用群聊 TTS | 开启 |
| **群聊白名单** | 仅对指定群聊生效（填写 UMO） | 空（全部生效） |
| **群聊触发概率** | 群聊中 LLM 回复转语音的概率 [0.0, 1.0] | 0.2 |
| **私聊语音开关** | 是否启用私聊 TTS | 开启 |
| **私聊白名单** | 仅对指定私聊生效（填写 UMO） | 空（全部生效） |
| **私聊触发概率** | 私聊中 LLM 回复转语音的概率 [0.0, 1.0] | 0.2 |

> **如何获取 UMO？** 在聊天中发送 `/sid`，即可看到当前会话的 UMO 信息。群聊格式为 `BotID:GroupMessage:群号`，私聊格式为 `BotID:FriendMessage:用户ID`，例如 `default:FriendMessage:114514`。将其填入对应的白名单即可。

### 翻译配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| **翻译开关** | 开启后 LLM 回复先翻译再合成 | 关闭 |
| **翻译模型** | 用于翻译的 LLM 模型 | 空 |
| **系统提示词** | 翻译指令 | 把下面的文本翻译成日语，不要额外解释 |
| **同时发送原文** | 语音 + 原文一起发送 | 关闭 |

## 指令用法（instruction）

通过指令控制方言、情感或角色等合成效果。长度限制 100 字符（汉字按 2 字符计算）。

### cosyvoice-v3.5-plus / cosyvoice-v3.5-flash

支持任意自然语言指令。这两个模型无系统音色，仅支持声音复刻/设计音色。

**示例**：

```
请用非常激昂且高亢的语气说话，表现出获得重大成功后的狂喜与激动。
语速请保持中等偏慢，语气要显得优雅、知性，给人以从容不迫的安心感。
语气要充满哀伤与怀念，带有轻微的鼻音，仿佛正在诉说一段令人心碎的往事。
请尝试用气声说话，音量极轻，营造出一种在耳边亲密低语的神秘感。
语气要显得非常急躁且不耐烦，语速加快，句子之间的停顿要尽量缩短。
请模拟一位慈祥、温和的长辈，语速平稳，声音中要透出满满的关怀与爱意。
语气要充满讽刺和不屑，在关键词上加重读音，句尾语调略微上扬。
请用一种极度恐惧且颤抖的声音说话。
语气要像专业的新闻播音员一样，冷静、客观且字正腔圆，情绪保持中立。
语气要显得活泼俏皮，带着明显的笑意，让声音听起来充满朝气与阳光。
```

### cosyvoice-v3-flash（复刻音色）

支持任意自然语言控制，包括方言、情感、语速等。

**方言**：

```
请用广东话表达。
```

支持：广东话、东北话、甘肃话、贵州话、河南话、湖北话、江西话、闽南话、宁夏话、山西话、陕西话、山东话、上海话、四川话、天津话、云南话。

**情感/语调**：

```
请非常生气地说一句话。
请非常开心地说一句话。
请非常恐惧地说一句话。
请非常伤心地说一句话。
请非常惊讶地说一句话。
请尝试一下亲和的语调。
请用冷酷的语调讲话。
请用威严的语调讲话。
用自然亲切的闲聊风格叙述。
用广播剧博客主的语气讲话。
```

### cosyvoice-v3-flash（系统音色）

指令必须使用固定格式和内容，详情请参见[音色列表](https://help.aliyun.com/zh/model-studio/cosyvoice-voice-list)。
